Fledgling

MCP tools that help AI agents get their bearings in a codebase, powered by DuckDB.

Install

curl -sL https://teaguesterling.github.io/fledgling/install.sql | duckdb

Run from your project root. Creates three files and you're done.

Requires duckdb CLI (v1.4+). Install: brew install duckdb or see duckdb.org.

What you get

  • .fledgling-init.sql — self-contained DuckDB init with all modules
  • .fledgling-help.md — skill guide for the agent
  • .mcp.json — MCP server config (merged with existing)

Tools

ReadLines File content with line ranges, context, and match filtering
FindDefinitions AST-based search for functions, classes, variables (27 languages)
CodeStructure Top-level structural overview of source files
GitDiffSummary File-level change summary between revisions
MDSection Read markdown sections by ID
ChatSessions Browse and analyze Claude Code conversation logs

Plus 20+ composable SQL macros available via the query tool.

SQL macros, not bash

-- Instead of: grep -rn "def parse" src/ SELECT * FROM find_definitions('src/**/*.py', '%parse%'); -- Cross-tier: definitions in recently changed files SELECT * FROM changed_function_summary('HEAD~5', 'HEAD', '**/*.py');

Customize

# Select specific modules and profile curl -sL https://teaguesterling.github.io/fledgling/install.sql \ | duckdb -cmd "SET VARIABLE fledgling_config = { modules: ['source', 'code', 'repo'], profile: 'core' }"

Available modules: source, code, docs, repo, structural, conversations, help.

Profiles: analyst (default, includes SQL query tool) or core (structured tools only).
